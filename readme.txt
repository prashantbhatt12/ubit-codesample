Backend:
1. to start up the node server go to the backend folder and open the same in a code editor.
2. open a terminal in the code editor and run "npm -init"
3. after this run "npm i cors pg express"
4. after run "node app.js" and make sure that the application is running on port 3001

Frontend:
1. to start the front end open the front end folder in a code editor
2. open up a new terminal inside the code editor.
3. run "npm install" to install all the packages.
4. run "npm run start"
